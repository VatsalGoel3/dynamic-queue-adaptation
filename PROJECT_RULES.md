# Project Rules

## One-Sentence Goal

Build a small offline prototype that tests whether manual queue insertions are useful short-term intent signals and whether reranking with them beats a static seed-only queue.

## What Will Be Built

- A Python-only offline recommender simulation
- A seed-only baseline recommender
- A queue-adaptive reranker
- One end-to-end dataset or synthetic-session setup
- Evaluation with 3 to 5 metrics, charts, and a concise writeup

## What Will Not Be Built

- Frontend
- Streaming app
- Browser extension
- Spotify, Apple Music, or YouTube API integration in V1
- Production deployment
- Deep learning training pipeline
- Any broader "playlist intelligence platform"

## Success Criteria

- Seed-only baseline recommender implemented
- Queue-adaptive reranker implemented
- At least one dataset or simulation setup runs end to end
- 3 to 5 meaningful metrics reported
- Tests pass
- README explains problem, approach, results, and limitations

## Stop Conditions

Stop and ship once all success criteria are true and the repo includes a readable writeup plus evaluation outputs.

## Time Budget

Target total effort: 12 to 20 focused hours.

If work meaningfully exceeds that budget, cut scope instead of adding features.
